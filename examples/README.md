# Template examples

These static HTML files are lightweight reference templates for teams that want page-level guidance without Storybook setup friction.

## Included templates

- `app-shell.html`: Authenticated product shell (header + nav + content + utility rail + footer).
- `auth-template.html`: Sign-in shell with benefit rail and form area.
- `marketing-template.html`: Hero + features + pricing marketing page.

Open any file directly in a browser, or serve from the repo root with a static file server.
